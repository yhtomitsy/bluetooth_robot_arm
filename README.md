bluetooth_robot_arm
===================

A bluetooth controlled arduino robot arm

This is a simple 6 dof arduino robot arm project that is controlled via bluetooth
What you need for this project is
-	An Arduino board (preferably an uno or due)
-	SSC-32 servo motor driver
-	HC-06 Bluetooth module
-	Power source (you can use batteries or a 5v power supply)
-	6-DOF robot arm (you can get one on www.bizoner.com)

The tutorial can be found on http://www.arduino-hacks.com/bluetooth-controlled-arduino-robot-arm/
